PTP::Debug

class PTP::Debug;

Application debug support.

Notes

_DEBUG should be defined to enable debug support. Debug replaces the global new and delete functions.


PTP::Debug::Dump

static void Dump (const void * data,
                  int size);

     data :  Data buffer.
     size :  Data size or -1 if data is a string (default).

Print contents of buffer in a user-readable form.

Notes

The data is output in both hex and ASCII (for printable characters).


PTP::Debug::Log

static void Log (const char * fmt,
                 ...);

     fmt :  printf style format string.
     ... :  Additional arguments if any.

Write a message to the debug log.


PTP::Debug::Scan

void Scan ();

Scan the heap for memory errors.


PTP::Debug::Error

static void Error (void * ptr);

     ptr :  Memory location.

Error is called when a memory error occurs.

Notes

This function is commonly used as a debugger hook.
